Digital management for your firearms collection and range sessions.

## Features
- Gun Inventory (Make, Model, Caliber, Serial)
- Range Session Logs (Round count, Targets)
- Maintenance Scheduling

## Stack
- FastAPI
- SQLite
- Docker

## Run
```bash
docker compose up -d --build
```
Access at `http://localhost:8003` (or `/armory` via portal proxy).
